# Komendy do ręcznego wrzucenia na GitHub

Uruchom te komendy w terminalu w folderze projektu:

```bash
# 1. Sprawdź status
git status

# 2. Dodaj wszystkie pliki
git add .

# 3. Zrób commit
git commit -m "Initial commit: Infinity Tech Landing Page with ReactBits"

# 4. Utwórz branch main (jeśli nie ma)
git branch -M main

# 5. Dodaj remote (zmień Story91 na Twoją nazwę użytkownika)
git remote add origin https://github.com/Story91/infinity-landing.git

# 6. Wrzut na GitHub
git push -u origin main
```

**Uwaga:** Zmień `Story91` w komendzie na swoją nazwę użytkownika GitHub!
