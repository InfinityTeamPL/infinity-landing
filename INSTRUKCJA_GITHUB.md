# Instrukcja wrzucenia na GitHub

Ponieważ GitHub CLI nie jest zalogowane, wykonaj te kroki ręcznie:

## Krok 1: Otwórz terminal w folderze projektu
```powershell
cd "c:\Users\Cezary\Documents\Infinity Tech\Infinity Landing"
```

## Krok 2: Utwórz repozytorium na GitHub.com
1. Wejdź na https://github.com/new
2. Nazwa repozytorium: `infinity-tech-landing`
3. Opis: `Infinity Tech - AI Solutions Landing Page with ReactBits`
4. Wybierz: **Public**
5. Kliknij: **Create repository**

## Krok 3: Skopiuj i wklej te komendy w terminalu:

```powershell
# Dodaj remote z Twoim username GitHub
git remote add origin https://github.com/Story91/infinity-tech-landing.git

# Wrzut na GitHub
git push -u origin main
```

## Gotowe!

Po wrzucie repozytorium będzie dostępne pod:
`https://github.com/Story91/infinity-tech-landing`
